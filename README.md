AeroFS Python SDK
==

*Work in progress.*

*Not officially endorsed by AeroFS at this time.*

The `aerofs-python-sdk` project is an AeroFS API SDK written in Python.

Install
---

```
virtualenv env
./env/bin/pip install -r requirements.txt 
```
